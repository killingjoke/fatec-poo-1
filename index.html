FATEC-POO... do que se trata?

FATEC-POO é um projeto open-source mas fechado para alguns colaboradores por enquanto pois se trata de um trabalho acadêmico de alguns alunos da matéria de Programação Orientada a Objetos da Faculdade de Tecnologia da Zona Leste, São Paulo - SP. A ideia do projeto é explorar/desenvolver o domínio sobre a orientação a objetos e os design patterns utilizando a linguagem de programação Java, que trabalha sobre tal paradigma.

Qual o problema a ser solucionado?

Atualmente os processos para o lançamento de notas escolares na maioria das escolas do Estado de São Paulo são manuais. Há algumas ferramentas disponíveis no mercado, mas ou são comerciais e caras para o uso no ramo público ou são complicadas de se usar.

Solução proposta

Desenvolver um sistema utilizando a linguagem multi-plataforma Java, com o uso de interface gráfica Swing, uma solução livre e de código-fonte aberto para futuras alterações, com o objetivo de automatizar operações manuais.

Como obtê-lo?

Para obter o código-fonte do projeto basta instalar a ferramenta git em seu computador (sistemas do tipo GNU/Linux geralmente oferecem a ferramenta por padrão) que pode ser obtida por aqui http://git-scm.com/ ou via repositório (em distros que fazem o uso de gerenciadores de pacotes). Após instalação, basta abrir o bash do git e digitar os seguintes comandos:

$ mkdir pasta_para_o_projeto
$ cd pasta_para_o_projeto
$ git clone https://github.com/leandrw/fatec-poo.git

Pronto! O projeto está em seu computador e pronto para ser usado.

Mas vamos direto ao ponto... quero usá-lo no Eclipse!?!

Para utilizar o repositório do Git no Eclipse é bem simples (info retirada de http://eclipse.github.com/#help ):

Menu Help > Install New Software. Escolha o site das atualizações do Eclispe e pesquise por 'git' na caixa de filtro. Após ter selecionado EGit, JGit, e Mylyn GitHub clique em Next para finalizar a instalação.

Se algum erro de atrito ocorrer durante o processo, basta ler com atenção as mensagens de erro e desmarcar os itens conflitantes um-a-um.
